# FirstProjects

Meus primeiros projetos em programação - estudos e experimentos iniciais

## Projetos

### Calculadora Java
Uma calculadora simples em Java com operações básicas:
- Adição
- Subtração  
- Multiplicação
- Divisão (com tratamento de divisão por zero)

**Como executar:**
```bash
javac Calculadora/src/main/java/Calculadora.java
java -cp Calculadora/src/main/java Calculadora
```
### Conversor de Temperatura
Um conversor simples de temperaturas entre Celsius e Fahrenheit com interface de linha de comando.

**Funcionalidades:**
- Conversão de Celsius para Fahrenheit
- Conversão de Fahrenheit para Celsius  
- Validação de entrada do usuário
- Interface simples e intuitiva

**Fórmulas utilizadas:**
- **Celsius → Fahrenheit:** `(°C × 9/5) + 32`
- **Fahrenheit → Celsius:** `(°F - 32) × 5/9`

**Como executar:**
```bash
javac ConversorTemperatura/src/main/java/ConversorTemperatura.java
java -cp ConversorTemperatura/src/main/java ConversorTemperatura
```
